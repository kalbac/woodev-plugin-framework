# Framework runtime autoloader + plugin type via inheritance — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Introduce a hand-written `spl_autoload` for framework classes so a plugin's type is declared solely by `extends`, and remove the now-redundant `capabilities` array.

**Architecture:** A single generated classmap (`woodev/class-map.php`) maps every framework class/interface/trait FQCN → file. `Woodev_Framework_Autoloader` registers an `spl_autoload_register` callback against the **winning** framework copy's path inside `Framework_Resolver::load_plugins()`, right after the winner is known. This makes `extends Woodev_Payment_Gateway_Plugin` resolve without a `capabilities` hint, so `load_early_capability_classes()` and the `capabilities` field are deleted. A thin `Woodev_Loader::register()` facade centralizes the entry-file boilerplate (framework-dir resolution, bootstrap require, B-1 mixed-fleet probe).

**Tech Stack:** PHP 7.4+, WordPress, Brain Monkey + Mockery (unit), PHPUnit, WPCS/PHPStan L3.

**Spec:** `docs-internal/specs/2026-06-21-plugin-type-autoloader-design.md`

**Conventions (project gotchas):** built-in `Edit` on existing PHP (NOT Serena `replace_content` — EOL flip); `@since 2.0.2`; Yoda conditions; short arrays; type declarations; `composer check` green after each task; never reintroduce a PHPStan baseline. Commit per task (Conventional Commits).

- [ ] **Step 1: Write the generator**

```php
<?php
/**
 * Dev tool: regenerate woodev/class-map.php by scanning the framework tree.
 *
 * Usage: php bin/generate-class-map.php
 * NOT shipped — lives in bin/, never required at runtime.
 *
 * @package Woodev\Framework
 */

declare( strict_types=1 );

$root      = dirname( __DIR__ );
$framework = $root . '/woodev';

// Files loaded explicitly (bootstrap / loader / the autoloader+map themselves) — never autoloaded.
$skip = [
	'woodev/bootstrap.php',
	'woodev/loader.php',
	'woodev/class-map.php',
	'woodev/class-framework-autoloader.php',
];

$map = [];

$iterator = new RecursiveIteratorIterator(
	new RecursiveDirectoryIterator( $framework, FilesystemIterator::SKIP_DOTS )
);

/** @var SplFileInfo $file */
foreach ( $iterator as $file ) {
	if ( 'php' !== strtolower( $file->getExtension() ) ) {
		continue;
	}

	$relative = str_replace( '\\', '/', substr( $file->getPathname(), strlen( $root ) + 1 ) );

	if ( in_array( $relative, $skip, true ) ) {
		continue;
	}

	$symbols = woodev_extract_symbols( (string) file_get_contents( $file->getPathname() ) );

	foreach ( $symbols as $fqcn ) {
		if ( isset( $map[ $fqcn ] ) ) {
			fwrite( STDERR, sprintf( "DUPLICATE class %s in %s and %s\n", $fqcn, $map[ $fqcn ], $relative ) );
			exit( 1 );
		}
		$map[ $fqcn ] = $relative;
	}
}

ksort( $map );

$export = "<?php\n";
$export .= "/**\n * GENERATED by bin/generate-class-map.php — do not edit by hand.\n";
$export .= " * Maps framework class/interface/trait FQCN => path relative to the framework copy root.\n";
$export .= " *\n * @package Woodev\\Framework\n */\n\n";
$export .= "return [\n";
foreach ( $map as $fqcn => $path ) {
	$export .= sprintf( "\t%s => %s,\n", var_export( $fqcn, true ), var_export( $path, true ) );
}
$export .= "];\n";

file_put_contents( $framework . '/class-map.php', $export );

fwrite( STDOUT, sprintf( "Wrote %d entries to woodev/class-map.php\n", count( $map ) ) );

/**
 * Extracts fully-qualified class/interface/trait names from PHP source via tokens.
 *
 * @param string $source PHP source.
 * @return string[] FQCNs.
 */
function woodev_extract_symbols( string $source ): array {
	$tokens    = token_get_all( $source );
	$namespace = '';
	$symbols   = [];
	$count     = count( $tokens );

	for ( $i = 0; $i < $count; $i++ ) {
		$token = $tokens[ $i ];

		if ( is_array( $token ) && T_NAMESPACE === $token[0] ) {
			$namespace = '';
			for ( $j = $i + 1; $j < $count; $j++ ) {
				$next = $tokens[ $j ];
				if ( ';' === $next || '{' === $next ) {
					break;
				}
				if ( is_array( $next ) && in_array( $next[0], [ T_STRING, T_NS_SEPARATOR ], true ) ) {
					$namespace .= $next[1];
				}
				if ( is_array( $next ) && defined( 'T_NAME_QUALIFIED' ) && T_NAME_QUALIFIED === $next[0] ) {
					$namespace .= $next[1];
				}
			}
			continue;
		}

		if ( is_array( $token ) && in_array( $token[0], [ T_CLASS, T_INTERFACE, T_TRAIT ], true ) ) {
			// Skip anonymous classes and `::class`.
			$prev = $tokens[ $i - 1 ] ?? null;
			if ( is_array( $prev ) && T_DOUBLE_COLON === $prev[0] ) {
				continue;
			}
			for ( $j = $i + 1; $j < $count; $j++ ) {
				$next = $tokens[ $j ];
				if ( is_array( $next ) && T_WHITESPACE === $next[0] ) {
					continue;
				}
				if ( is_array( $next ) && T_STRING === $next[0] ) {
					$symbols[] = '' !== $namespace ? $namespace . '\\' . $next[1] : $next[1];
				}
				break;
			}
		}
	}

	return $symbols;
}
```

- [ ] **Step 1: Write the autoloader**

```php
<?php
/**
 * Framework runtime autoloader.
 *
 * @package Woodev\Framework
 */

defined( 'ABSPATH' ) || exit;

if ( ! class_exists( 'Woodev_Framework_Autoloader', false ) ) :

	/**
	 * Hand-written spl_autoload for framework classes (no Composer in shipped plugins).
	 *
	 * Owns only classes present in the generated classmap, resolved against the selected
	 * framework copy's root. Returns silently for anything else so other autoloaders run.
	 *
	 * @since 2.0.2
	 */
	final class Woodev_Framework_Autoloader {

		/** @var bool Whether the autoloader has been registered. */
		private static bool $registered = false;

		/** @var string Absolute path to the selected framework copy root (the dir containing woodev/). */
		private static string $base_path = '';

		/** @var array<string,string> FQCN => relative path map. */
		private static array $map = [];

		/**
		 * Registers the autoloader against the selected framework copy.
		 *
		 * @since 2.0.2
		 *
		 * @param string $base_path Absolute path to the dir that contains the `woodev/` folder.
		 * @return void
		 */
		public static function register( string $base_path ): void {
			if ( self::$registered ) {
				return;
			}

			self::$base_path = rtrim( $base_path, '/\\' );
			self::$map       = (array) require self::$base_path . '/woodev/class-map.php';
			self::$registered = true;

			spl_autoload_register( [ self::class, 'autoload' ] );
		}

		/**
		 * Resolves a framework class to its file and loads it.
		 *
		 * @since 2.0.2
		 *
		 * @param string $class Fully-qualified class name.
		 * @return void
		 */
		public static function autoload( string $class ): void {
			$relative = self::$map[ $class ] ?? null;

			if ( null === $relative ) {
				return;
			}

			$file = self::$base_path . '/' . $relative;

			if ( is_readable( $file ) ) {
				require_once $file;
			}
		}

		/**
		 * Resets state. Test-only.
		 *
		 * @since 2.0.2
		 *
		 * @return void
		 */
		public static function reset(): void {
			if ( self::$registered ) {
				spl_autoload_unregister( [ self::class, 'autoload' ] );
			}
			self::$registered = false;
			self::$base_path  = '';
			self::$map        = [];
		}
	}

endif;
```

- [ ] **Step 1: Write the facade**

```php
<?php
/**
 * Thin plugin entry facade.
 *
 * @package Woodev\Framework
 */

defined( 'ABSPATH' ) || exit;

if ( ! class_exists( 'Woodev_Loader', false ) ) :

	/**
	 * Centralizes plugin entry boilerplate: framework-dir resolution, bootstrap require,
	 * the B-1 mixed-fleet probe, and loader-definition registration.
	 *
	 * @since 2.0.2
	 */
	final class Woodev_Loader {

		/**
		 * Registers a plugin with the framework bootstrap.
		 *
		 * @since 2.0.2
		 *
		 * @param string              $plugin_file The plugin entry __FILE__.
		 * @param array<string,mixed> $definition  Loader definition (without 'plugin_file').
		 * @return bool True when registered; false when dormant (legacy copy won) or invalid.
		 */
		public static function register( string $plugin_file, array $definition ): bool {
			$framework_dir = defined( 'WOODEV_FRAMEWORK_DIR' )
				? (string) WOODEV_FRAMEWORK_DIR
				: dirname( $plugin_file ) . '/woodev-framework';

			// Allow a plugin to bundle the framework under its own /woodev directly.
			$bootstrap = is_readable( $framework_dir . '/woodev/bootstrap.php' )
				? $framework_dir . '/woodev/bootstrap.php'
				: dirname( $plugin_file ) . '/woodev/bootstrap.php';

			if ( ! is_readable( $bootstrap ) ) {
				return false;
			}

			if ( ! class_exists( 'Woodev_Plugin_Bootstrap', false ) ) {
				require_once $bootstrap;
			}

			$instance = \Woodev_Plugin_Bootstrap::instance();

			// B-1 mixed-fleet probe: a legacy v1 copy won the class rendezvous.
			if ( ! method_exists( $instance, 'register_loader_definition' ) ) {
				return false;
			}

			$definition['plugin_file'] = $plugin_file;

			return $instance->register_loader_definition( $definition );
		}
	}

endif;
```

> The facade does NOT render the B-1 admin notice itself (different plugins want different
> wording, and the notice path must avoid framework classes since a legacy copy is loaded).
> It returns `false` so the entry file can render its own notice — exactly as the fixtures do
> today. This keeps the facade dependency-free and the notice plugin-owned.
